--INSTRUCCIONES PARA INSTALAR EL APLICATIVO:

	1. Se descomprime el archivo .rar llamado "prueba_konecta"
	2. Se copia la carpeta que contiene el proyecto llamada "Konecta-Cafe" y se pega dentro de htdocs
	3. Se crea una BD en phpmyadmin y se importa la bd llamada bd_restaurante
	3. Se ejecuta el programa por medio del localhost

--OBSERVACIONES:
	1. En caso de que el inicio de sesion falle, validar que el aplicativo se esté ejecutando bajo el archivo page.php
	2. En caso tal de que no se elimine algun registro, será porque este es llave foranea en alguna otra tabla
